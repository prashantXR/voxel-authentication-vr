# Voxel Authentication for Virtual Reality

This project explores **voxel-based graphical passwords for Virtual Reality (VR)**, with a focus on **secure, usable, and privacy-aware authentication** in immersive environments.

It builds on our work on **VoxAuth**, a voxel-based 3D authentication approach using **eye-gaze input**, and our later evaluation of **2D and 3D voxel-based graphical passwords** in VR.

---

## Overview

As VR becomes increasingly social and mobile, users need authentication methods that are not only secure, but also practical for immersive use. Traditional input methods may expose sensitive actions through visible body movement or controller interaction.

This project investigates voxel-based password entry in VR as an alternative that supports memorability and structured selection while reducing observation risks in immersive settings.

---

## Research Questions

- How usable are voxel-based graphical passwords in VR?
- How do **2D** and **3D** voxel passwords compare in performance?
- What design trade-offs emerge between **security, memorability, and ease of use** in immersive authentication?

---

## Key Contributions

- Evaluated **voxel-based graphical passwords** for authentication in Virtual Reality
- Compared **2D and 3D password structures** under different difficulty conditions
- Identified usability differences in terms of **completion time, accuracy, and error rate**
- Extended prior work on **VoxAuth**, which used **eye-gaze** for private password input in social VR contexts

---

## Main Findings

Publicly available summaries of the 2024 study report that:

- **2D voxel-based graphical passwords** were easier to use than 3D versions
- 2D passwords led to **faster completion times**
- 2D passwords achieved **higher selection accuracy**
- 2D passwords produced **lower error rates**
- Participants generally preferred the 2D approach over the 3D one

These findings suggest that while 3D passwords may appear appealing in immersive environments, simpler 2D structures may offer a better balance of usability and practicality for current VR authentication systems.
