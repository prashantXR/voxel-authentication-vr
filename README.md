# Voxel Authentication for Virtual Reality (IEEE VRW 2024)

This project investigates **voxel-based graphical passwords for authentication in Virtual Reality**, focusing on **usability, memorability, and privacy** in immersive environments.

📍 Published at **IEEE VR Workshop (VRW) 2024**  
📍 Authors: Prashant Rawat, Rumeysa Türkmen, Chukwuemeka Nwagu, Kissinger Sunday, Mayra Donaji Barrera Machuca  

---

## 🧠 Motivation

As Virtual Reality becomes increasingly social and interactive, traditional authentication methods become difficult to apply due to visibility, input constraints, and privacy concerns.

This work explores how **voxel-based graphical passwords** can provide a **structured and immersive authentication mechanism**, while reducing risks such as observation and input exposure in VR environments.

---

## 🔬 Contributions

- Designed and evaluated **voxel-based graphical passwords** for VR authentication  
- Compared **2D and 3D voxel password structures** under different task conditions  
- Analyzed performance in terms of **completion time, accuracy, and error rates**  
- Extended prior work on **VoxAuth**, a gaze-based authentication approach in VR

## 🧪 Method Overview

- User study conducted in immersive Virtual Reality  
- Participants created and entered **voxel-based graphical passwords**  
- Compared **2D vs 3D password configurations**  
- Measured usability metrics including **time, accuracy, and error rate**  
- Evaluated participant preferences and interaction challenges
